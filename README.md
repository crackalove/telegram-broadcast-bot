# Telegram Broadcaster (Telethon)

Авторассылка сообщений в Telegram через вашу сессию Telethon.

## Возможности
- Отправка сообщений каждые N минут (интервал задаётся в `config.json`)
- Список чатов/каналов (username или `chat_id`)
- Список сообщений (рандомный выбор каждого отправления)
- Рандомизация интервала (±10% по умолчанию) для снижения риска бана
- Логирование в файл и консоль
- Обработка FloodWait и других ошибок
- Остановка скрипта по Ctrl+C
- Статистика отправок в логах

## Установка
```bash
pip install -r requirements.txt
```

## Конфигурация
Заполните `config.json`. (api_id, api_hash, interval, messages, chats)

## Запуск
```bash
python main.py
```
При первом запуске Telethon попросит ввести код подтверждения для вашей сессии.

## Настройки
- `interval_minutes`: базовый интервал отправки в минутах.
- `jitter_ratio`: доля рандомизации (0.1 = ±10%).
- `chat_list`: username (с `@`) или `chat_id`.
- `messages`: список строк, выбирается случайно при каждой отправке. (если одно сообщение, следовательно, будет отправляться только оно)

## Примечания
- FloodWait обрабатывается автоматически: скрипт подождёт нужное время.
- Логи пишутся в `send.log` и выводятся в консоль.
- Для завершения нажмите Ctrl+C — скрипт отключится.
